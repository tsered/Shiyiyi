# 定义的函数

## Description

定义函数 `f(x)` 表示 `x` 的各位数字之和，比如 `f(1)=1, f(55)=10, f(345)=12`.

现在给定 `常数A` 和 `区间[L, R]`，问你在这个区间里面有多少个数 `x` 满足 `f(x)*f(x)=f(A*x)`.

## Input

不超过 40 组数据。一组数据一行，包括三个整数 `A，L，R`。数据范围：`0<=A<100， 1<=L<=R<=1000,000,000`。

## Output

首先输出 case 数，然后是答案。

## Sample Input

```
1 1 10
2 1 10
88 888 8888
```

## Sample Output

```
Case 1: 2
Case 2: 1
Case 3: 0
```
