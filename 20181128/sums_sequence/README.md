# 和数数列

## Problem Description

一个数列 `1,3,6,10,15,21,…,` 每一个数都是从1到该项序的和数。

例如，第五项的值15为从1加到5的和。

现在对于每个给定的项数`n`，计算该数列从第`1`项到第`n`项的和。

例如，给定项值5，则从第1项到第5项的和为35。

## Input

有一些整数`n(0<n<2344)`。

## Output

输出n所对应的从第1项到n项的和数列的和。每个输出占一行。

## Sample Input

```
3 5
```

## Sample Output

```
10
35
```
