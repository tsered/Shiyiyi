# 12！配对

## Description

找出输入数据中所有两两相乘的积为12!的个数。

## Input

输入数据中含有一些整数`n（1≤n＜2^32）`。

## Output

输出所有两两相乘的积为12!的个数。

## Sample Input

```
1 10000 159667200 9696 38373635
1000000 479001600 3
```

## Sample Output

```
2
```
