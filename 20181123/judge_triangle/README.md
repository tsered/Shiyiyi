# 判断三角形

## Problem Description

给定3个整数，A，B，C<=100，请判断能否组成一个三角形。

## Input

输入第一行为整数N，表示一共有N组数据。接下来N行每行三个整数Ai，Bi，Ci。

## Output

输出N行，如果第i组数据中的ABC能组成一个三角形请输出”YES”（大写，不带引号），否则输出“NO”。

## Sample Input

```
4
1 2 3
100 0 100
100 100 1
1 1 1
```

## Sample Output

```
NO
NO
YES
YES
```
