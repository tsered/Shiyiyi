# 比大小

## Description

阳阳爱玩一个叫比大小的游戏，不过按老师教的方法比大小太没意思了，于是他自己想了另一个方法来比较两个整数的大小： 

-位数少的数比较大 
-两个数位数相等时，各位数字之和小的比较大 
-若两数位数、各位数之和都相同，则判定两数相等 

你能比阳阳更快知道哪个数更大吗？

## Input

输入包含多组数据，每组数据包含两个整数`a，b（0<=a,b<2^32）`，处理到文件结束。

## Output

每行输出较大的数，若两数相等，则输出"the same"。

## Sample Input

```
12 3
528 199
1561 6511
```

## Sample Output

```
3
528
the same
```
