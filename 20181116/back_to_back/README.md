# 背靠背

## Problem Description

根据读入的字符和高，勾画背靠背字符三角形。

## Input

输入数据含有不超过50组的数据，每组数据包括一个可见字符c和一个整数n（1≤n≤30）。

## Output

输出以c为填充字符，高为n的背靠背字符三角形，勾画每个三角形时都应另起一行。

## Sample Input

```
W 5 B 3
```

## Sample Output

```
    W W
   WW WW
  WWW WWW
 WWWW WWWW
WWWWW WWWWW
  B B
 BB BB
BBB BBB
```
