# 英文 A+B

## Description

输入两个小于100的正整数A和B，计算A+B。注意：A和B的每一个数字由对应的英文单词组成。

## Input

测试输入包含若干测试用例，每个测试用例占一行，格式为"A + B ="，相邻两字符串有一个空格间隔。当A和B同时为zero时输入结束，相应的结果不要输出.

## Output

对每个测试用例输出1行，即A+B的值.

## Sample Input

```
one + two =
three four + five six =
seven + eight nine =
one two + two zero =
zero + zero =
```

## Sample Output

```
3
90
96
32
```
