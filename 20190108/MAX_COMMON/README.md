# 最大公共字符集

## Description

将两个字串中都出现的字符全部挑出来，就构成最大公共字符集了。

## Input

多组数据。 每组数据有两个字串(字串长度<100)。字串由没有空格的可见ASCII字符集中的字符所构成。字串中没有重复字符。

## Output

对于每组数据，以一行的方式输出一个最大公共字符集字串，该字串中的字符应以ASCII值从小到大的顺序排列。如果没有公共字符，则输出 `NULL`。

## Sample Input

```
asdf
gdfh
ABCDXE
XYZABCD
A
B
```

## Sample Output

```
df
ABCDX
NULL
```
