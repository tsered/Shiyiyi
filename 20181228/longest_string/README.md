# 最长的字串

## Description

给你两个字符串A和B（只包括26个小写字母），你的任务是求出最长的一个子串S。其中S是A的后缀且是B的前缀。只要给出S的最大长度就可。 

PS：后缀是指S包含字符串的最后一个字符，如“34”是“25234”的后缀。前缀是指S包含字符串第一个字符，如“34”是“34523”的前缀。

## Input

多组测试数据，处理到文件结束。 每组数据包括两个字符串A和B（字符串长度小于等于100000），分别在两行给出。

## Output

对于每个测试数据，输出S的最大长度。

## Sample Input

```
aabbc
bbcaa
aaa
aaa
abcd
efgh
```

## Sample Output

```
3
3
0
```
