# 打字

## Description

笨笨的chen_h 最近在练习打字。当他把手放在键盘上时，稍不注意就会往右错一位。这样的话，A会变成S，I会变成O等，但空格还是空格。简化后的电脑键盘为：

```
` 1 2 3 4 5 6 7 8 9 0 - =
Q W E R T Y U I O P [ ]
A S D F G H J K L ; ‘ \
Z X C V B N M , . /
```

## Input

输入数据有多组，每组输入一个错位后敲出的句子（所以输入的字母都是大写），处理到文本结束。

## Output

输出 chen_h 本来想打出的句子。注意：键盘最左边的字符不会错位，例如，敲Q与W都敲的是Q。

## Sample Input

```
O ;OLR UPI/
O S, GOMR YPFSU/
```

## Sample Output

```
I LIKE YOU.
I AM FINE TODAY.
```
