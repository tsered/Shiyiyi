# 交替字符倒三角形

## Description

根据读入的高，勾画 `ST` 字符交替的倒三角形。

## Input

输入数据含有不超过50个正整数 `n（1≤n≤30）`。

## Output

输出以 `n` 为高的 `ST` 字符交替的倒三角形，勾画每个三角形时都应另起一行。

## Sample Input

```
3 10
```

## Sample Output

```
STSTS
 STS
  S
STSTSTSTSTSTSTSTSTS
 STSTSTSTSTSTSTSTS
  STSTSTSTSTSTSTS
   STSTSTSTSTSTS
    STSTSTSTSTS
     STSTSTSTS
      STSTSTS
       STSTS
        STS
         S
```
