# 按日期排序

## Description

有一些日期，日期格式为“MM/DD/YYYY”。编程将其按日期大小排列。

## Sample Input

```
12/31/2005
10/21/2003
02/12/2004
15/12/1999
10/22/2003
11/30/2005
```

## Sample Output

```
15/12/1999
10/21/2003
10/22/2003
02/12/2004
11/30/2005
12/31/2005
```
