# 门帘设计

## Problem Description

小明家新装修，想做一个门帘，先在图纸上画了几个方案，都不理想，看来用计算机来帮她是最好的了。请根据输入数据，画出若干门帘图案供小明挑选。

## Input

每组数据包括门帘图案，门幅，以及装饰条长度。宽度一律为29。

## Output

输出门帘图案，上下图案之间空一行，最后图案后面没有多余的空行。

## Sample Input

```
W 10 3
H 11 8
```
