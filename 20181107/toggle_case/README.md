# 字母转换

## Problem Description

将字母的大小写转换一下，非字母字符原样输出。
 
## Input

一行文本。

## Output

转换大小写输出。

## Sample Input

```
I like ACM/ICPC.
Hello World!
+-*/ are operators.
```

## Sample Output

```
i LIKE acm/icpc.
hELLO wORLD!
+-*/ ARE OPERATORS.
```
