# 可简约素因子

## Description

给定一个整数，判断其是否素因子可简约。一个整数是由若干个素因子的积合成的，如果其中一个素因子的幂次方不少于2，则称该整数的素因子可简约。

例如：24=2^3X3,所以24是素因子可简约的。42=2X3X7，所以42的素因子是不可简约的。

## Input

有一些正整数(<2^31)，若为0，则输入结束。

## Output

对应每个整数，如果素因子可简约，则输出Yes，否则输出No。

## Sample Input

```
24 42 0
```

## Sample Output

```
Yes
No
```
