# 注释

## Description

给出一个C++源程序代码。请将其中的注释去掉。

## Sample Input

```
//======================

// simplest program

//======================
#include<iostream>
using namespace std;
//----------------------
int main(){
cout<<”hello world!\n”;
}//---------------------
```

## Sample Output

```
#include<iostream>
using namespace std;
int main(){
cout<<”hello world!\n”;
}
```
