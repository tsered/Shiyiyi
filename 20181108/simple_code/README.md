# 简单编码

## Problem Description

最近Kingly对编码很感兴趣，于是从网上找了一些编码原则来对字符串做实验。因为Kingly一直很忙，所以希望你这位编程高手来替他解决这个问题。 下面是编码原则： （1） 如果访问到字符A，W，F就转化成I； （2） 如果访问到字符C，M，S就分别转化成L，o，v； （3） 如果访问到字符D，P，G，B就转化成e； （4） 如果访问到字符L，X就分别转化成Y，u； （5） 其他字符均保持不变； （6） 遇到END就结束！
